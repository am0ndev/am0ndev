### FujiTap - Industry Collaboration Android App

**Repository Name:** `FujiTap`

FujiTap is an Android tablet application developed through a university-industry collaboration project.
The project focused on solving real problems faced by a company by applying knowledge from information engineering and software development.

The application supports PDF-based drawing/document viewing and enables users to zoom, pan, and add comments directly within the app.
It was designed as a practical tool to support workflow improvement, information sharing, and technical knowledge transfer.

**Tech Stack:** Java, XML, Android Studio, Gradle, Android SDK

**Key Features**

* PDF drawing/document viewer
* Zoom in, zoom out, and reset view controls
* Drag-based panning for image/PDF viewing
* Comment panel for adding, editing, and deleting comments
* Internal PDF file handling using Android storage features
* Tablet-oriented UI layout support
* Developed based on a real company problem

**What I Learned**

* How to translate business problems into application features
* Android app development using Java and XML
* UI design for tablet-based workflows
* File handling and PDF rendering on Android
* The importance of designing software for actual users, not only for assignments
